Le Tardis est une projet qui permet de visualiser un tas de donnée pour faire des prédictions des futures déplacements des trains futures.


## Features

- Can parse csv file and clean this
- Can predict information with only one csv
- Dashboard for visual result with data csv

## Stack Use

- Python 3
- Jupiter NoteBook
- Streamlit dashboard

## Getting Started

### Installation and environement

For install the dependencies and start the project, just need :

```bash
 source ./Init_env.sh 
```

### Lauch the project 

Just lauch the program with :

```bash
./tardis.py
```

## Project Tree

This is a tree for this project :

```
├── requirment.txt      # All project dependencies
├── tardis_eda.ipynb    # Data cleaning, exploration, and feature engineering
├── cleaned_dataset.csv     #Processed dataset output from EDA notebook
├── tardis_model.ipynb Model    #training, evaluation, and selection
├── model.pkl or model.joblib   #Trained model file
├── tardis_dashboard.py     #Interactive Streamlit dashboard
├── README.md       #Installation, usage, and project documentation
```
